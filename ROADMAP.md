# Technical Implementation Roadmap for Cline

## Phase 1: Environment & Base Engine
- [ ] Initialize Next.js or Vite project with Tailwind CSS.
- [ ] Install Animation Libraries: `framer-motion`, `gsap`, and `lenis` (for smooth scrolling).
- [ ] Setup Global Styles: Implement the "Organic Minimalism" CSS variables.

## Phase 2: The "Wholesome" Animation Shell
- [ ] **Smooth Scroll:** Integrate Lenis for high-end "momentum" scrolling.
- [ ] **Page Transitions:** Create a "Fade & Slide" transition between routes.
- [ ] **Text Reveal:** Every heading must use a "Split-Type" animation where characters fade up individually on scroll.
- [ ] **Parallax Images:** Implement "Scale-Down" parallax (Images start at 1.2x scale and shrink to 1x as they enter the viewport).

## Phase 3: Content Engineering (The Detail Layer)
- [ ] **Product Catalog:** Build sections with technical specs:

    - *ETP/STP:* Focus on 1-500 KLD capacity and MBBR/SBR technology.
- [ ] **Client Grid:** A grayscale-to-color hover grid featuring Sir Ganga Ram Hospital, Sahara Hospital, and Pharmazz India.
- [ ] **Legal Footer:** Deep navy background with glassmorphism cards displaying GST and Udyam details.

## Phase 4: AI Image Generation Strategy
